  Simon Says Game
</h1>

<p align="center">
  An interactive memory-based game that challenges players to remember and repeat
  a growing sequence of colors.
</p>

<hr>

<h2>
  <img src="https://cdn-icons-png.flaticon.com/512/4712/4712109.png" width="28">
  Description
</h2>

<p>
  Simon Says is a classic memory game where each round adds a new color to the
  sequence. The player must correctly repeat the entire pattern to advance.
  As levels increase, the sequence becomes longer and faster, testing focus
  and memory skills.
</p>

<h2>
  <img src="https://cdn-icons-png.flaticon.com/512/2099/2099058.png" width="28">
  How It Works
</h2>

<ol>
  <li>Press the <strong>Start</strong> button to begin the game.</li>
  <li>The game flashes a random color.</li>
  <li>Repeat the sequence by clicking the color buttons in order.</li>
  <li>If correct, the next level starts with an extended pattern.</li>
  <li>If wrong, <strong>Game Over</strong> appears and the game can restart.</li>
</ol>

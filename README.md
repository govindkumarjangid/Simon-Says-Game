🎮 Simon Says Game

Simon Says is an interactive memory-based game that tests the player’s ability to remember and repeat sequences of colors or sounds. Each round, the game adds a new color to the sequence, and the player must repeat it correctly to advance. The sequence gets longer and faster with each level, challenging the player’s focus and memory skills.

🧠 Features

🔔 Color Sequence Memory: Follow the sequence of flashing colors.

⏫ Progressive Difficulty: Each level adds a new step to the pattern.

🎨 Vibrant Visuals: Animated color flashes and sound feedback for each button.

💥 Game Over Alert: Displays a red color and resets when the user makes a mistake.

⚡ Responsive Design: Works smoothly across all screen sizes.

🧩 Tech Stack

Frontend: HTML, CSS, JavaScript

Logic: DOM manipulation, event listeners, and random color generation

Sound Effects: Optional audio cues for better gameplay experience

🚀 How It Works

Press the start button to begin the game.

The game flashes a random color.

The player repeats the sequence by clicking the color buttons in order.

If correct — next level begins with an extended pattern.

If wrong — “Game Over” appears, and the player can restart.
